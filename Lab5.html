Concept of WWW

  • It stands for World wide web.
  • It is basically a system of internet servers that support specially formatted documents.
  • It is a collection of text pages,vidoes,digitdl photographs and animations we can access over the internet.
  • User access the World Wide Web facilities via client called a web browser

    HTTP protocol

  • HTTP stands for Hypertext transfer protocole
  • HTTP is based on the client-server architecture model and a stateless request/response protocol
  • 404 file not found is a common HTTP status code
